# Quote of the Day App

A full-stack web application that displays inspirational quotes from notable personalities. Users can generate a new random quote with a single click, while the backend serves quotes through REST API endpoints.

## Features

- Display random inspirational quotes
- Generate a new quote on button click
- Show quote text and author
- Backend API for quote retrieval
- Responsive and user-friendly interface
- Loading state while fetching data

## Tech Stack

  # Frontend
   - React.js
   - HTML5
   - CSS3
   - Javascript

  # Backend
   - Node.js
   - Express.js
